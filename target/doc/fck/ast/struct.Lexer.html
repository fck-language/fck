Lexer for fck

This works on a (8-bit) character by character basis. 16-bit characters are still in production so be patient

the current character the pointer is pointing to

Called by self.make_tokens when a ‘@’ character is found. This advances and calls self.make_identifier, taking the value of the token returned and placing it into a new TT_AT token

fn make_string(&mut self, starting_character: char) -> Result<Token, Error>

Called by self.make_tokens when an opening string delimiter is found. Parses the string and returns a TT_STRING token containing the string literal in an Ok(Token). Returns an Err(Error) if no closing delimiter is found.
